# 设计案例模板

添加新案例时使用此模板。

```markdown
---
name: 设计师姓名
name_en: Designer Name
discipline: Architecture | UX-UI | Product | Interior | Landscape | DesignEngineering
website: https://portfolio-url.com
location: 城市, 国家
location_en: City, Country
completion_year: 2024
tags: [tag1, tag2, tag3]
---

# 设计师姓名

## 简介
2-3句话介绍设计师和他们的设计方向

## 设计特点
- 设计风格
- 专注领域
- 技术栈/工具

## 代表作品

### 作品1: 作品名称
- **领域:** Architecture/UX/Product
- **链接:** [查看作品](url)
- **亮点:** 简述为什么值得学习

### 作品2: 作品名称
- **领域:** Architecture/UX/Product
- **链接:** [查看作品](url)
- **亮点:** 简述为什么值得学习

## 学习要点
- [ ] 可以学习的设计思路
- [ ] 值得参考的技术实现
- [ ] 设计亮点分析
```

## 字段说明

| 字段 | 说明 |
|------|------|
| name | 设计师姓名 |
| discipline | 设计领域 |
| website | 作品集网站 |
| location | 所在地 |
| tags | 标签，方便筛选 |
